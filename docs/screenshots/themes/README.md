# Theme gallery

Every bundled theme, captured live from the panel dashboard with mock data
(the screenshots rig — `.claude/skills/screenshots/capture-themes.mjs`).
Pick any of these under **Settings → Appearance**; it applies instantly.
See [docs/THEMING.md](../../THEMING.md) for how themes work and how to make one.
